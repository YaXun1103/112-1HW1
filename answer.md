## 說明程式與內容
要先複製一份到⾃⼰的專案資料集 git clone
git add 檔案名稱
git commit
將資料推送上去，並在伺服器倉庫確認是否推送成功 git push
git pull
新增分支 git branch 分支名稱
切換到新分支 git checkout 分支名稱
根據題目去新增檔案並且推送⾄伺服器倉庫
合併分支 git merge 分支名稱 合併完push
在命令選擇區輸入git log 可以看到分支的紀錄
